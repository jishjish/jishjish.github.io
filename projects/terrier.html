Terrier AI

Rust · Python · AST
terrierai.dev ↗

Automated pytest generation for Python developers. Built because writing tests is necessary but writing boilerplate is painful.

Why

Have you or someone you know been effected by vibe coding? A great idea that turned into an unweildy code base after one too many "make it better" prompts? Well you are not entitled to compensation, but you may get to know your project better with the help of Terrier. Understand the structure, functions, and sources of your repo through our straightforward CLI.

What it does

A background agent that watches your Python files, parses their ASTs, and generates pytest coverage in real time. Handles nested classes, edge cases, and fixture inference.

Stack

Rust core for file watching via the notify crate. AST parsing to understand function signatures and types. LLM integration for edge cases where static analysis falls short. GUI built with the Iced framework.
